# profile
Personal website focused on getting hired as a software engineer.

## Overview
This is a simple, modern, static site with placeholders you can replace with
your own content. It highlights your experience, projects, skills, and contact
information.

## Getting started
- Open `index.html` in your browser.
- Update the placeholder text (anything in brackets) with your details.
- Add a `resume.pdf` file if you want the Resume links to work.

## Structure
- `index.html`: Main content and layout.
- `styles.css`: Styling and responsive layout.
- `script.js`: Small enhancements (auto-updates the footer year).
- `AGENTS.md`: Guidance for AI agents working in this repository.
